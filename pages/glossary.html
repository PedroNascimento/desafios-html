Glossário da Programação

Algoritmo
Um algoritmo é uma fórmula ou conjuntos de etapas para resolver um problema específico.
Data Structure - Estrutura de Dados
Na programação, o termo estrutura de dados refere-se a um esquema para organizar informações relacionadas.
Decoder - Decodificar
Decodificação redere-se à reversão do processo de um método de codificação
Garbage Colletion
É um processo para a automação do gerenciamento de memória.
Kernel
É um módulo central de um sistema operacional (SO). É a parte do sistema operacional que carrega primeiro e permanece na memória principal
UML
Abreviação de Unified Modeling Language, uma linguagem notacional de uso geral para especificar e visualizar softwares complexos, especialmente grandes projetos orientados a objetos.
